# Image Swap Guide — Team Line Eco Resort

This guide lists all the unique `id` attributes used for placeholder images across the website. Replace the content of these `div`s with real client photos.

## Global / Home Page
| ID | Description |
|---|---|
| `hero-banner` | Main Hero background image for Home page |
| `amenity-pool` | Swimming Pool & Rain Dance photo |
| `amenity-cricket` | Box Cricket arena photo |
| `amenity-gym` | Open Gym photo |
| `amenity-projector` | Projector TV Area photo |
